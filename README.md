# 🦸‍♂️ Marvel T-Shirts Website (Dockerized)

## 🚀 Project Overview

This project is a Marvel-themed static e-commerce website showcasing T-shirts, built using HTML, CSS, and JavaScript. The application is containerized using Docker and served via Nginx for consistent and scalable deployment.

---

## 🎯 Features

* 🛍️ Product listing (Marvel-themed T-shirts)
* 🧾 Product cards with modern UI
* 🛒 Add-to-cart interaction (frontend)
* ⚡ Responsive design
* 🐳 Fully Dockerized for easy deployment

---

## 🧰 Tech Stack

* HTML5
* CSS3
* JavaScript
* Docker
* Nginx
